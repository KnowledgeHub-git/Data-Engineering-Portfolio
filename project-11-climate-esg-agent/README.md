# Project 11 — Climate & ESG Compliance Agent

AI agent that assesses company ESG compliance by combining emissions data, climate policy search, and company profiles to answer regulatory questions.

---

## Status: Planned

## Key Deliverables

- [ ] Agent tools: search_regulations, query_emissions, get_company_profile
- [ ] Use case: "Is Company X compliant with EU CSRD reporting based on their emissions?"
- [ ] Multi-step reasoning with evidence gathering

## How This Fits in the 15-Project Plan

Demonstrates an agent in a **regulated domain** (ESG/sustainability) — differentiates from the market research agent by showing compliance and audit patterns, which require factual accuracy and source citation.
